Rules of Code Retreat
=====================
- Write tests first
- Code in 2-tuples (or rarely 3-tuples)
- Stop in the middle of
- Delete code (except next to last session)


Rules of Simple Design
======================
- Passes all tests
- Express Intent
- No Duplication
- Minimal methods, classes, & modules
